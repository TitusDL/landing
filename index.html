🔗

연결

실행

할 일을 자동으로 처리합니다.

주요 기능 소개

  • 고객 정보 자동 정리 – 연도 기반 ID 생성 및 상태별 필터
  • 가족 정보 자동 연결 – 남편·아내 보험 가입 자동 연동
  • To-Do 기능 – 고객별 해야 할 일, 주간/일간 자동 분류
  • 생일 및 기념일 자동 알림 – 놓치지 않고 관계 유지
  • 상품별 정보 통합 뷰 – 8가지 보험을 하나의 뷰에서 확인
  • 업셀링 기회 제공 – Prospect 상태로 상품 추천 재접근

실제 활용 시나리오

  • 신규 상담: Prospect로 등록 → 상담 메모 남기기 → 추후 자동 알림
  • 기존 고객: 이름만 검색해도 가입 상품·가족 관계·메모 확인
  • 생일 알림: 오늘 또는 이번 주 생일 고객 자동 표시
  • 업셀링: 아직 가입 전인 상품 필드 통해 자연스러운 제안
